# App Predios Demo - Versión 3

## Instrucciones de Instalación

1. Descargue el archivo `AppPrediosDemo-Version3.zip`
2. Extraiga el contenido del archivo ZIP en una carpeta de su elección
3. Abra la carpeta extraída
4. Ejecute `AppPrediosDemo.exe`
5. Configure la conexión a la base de datos siguiendo las instrucciones en `CONFIGURACION_CONEXION.txt`

## Requisitos

- Windows 10 o superior
- No requiere instalación de .NET Framework (incluido en el ejecutable)

## Características

- Registro de nuevos predios
- Consulta y edición de registros existentes
- Interfaz moderna y profesional
- Validación de campos obligatorios
- Secciones colapsables para mejor organización
